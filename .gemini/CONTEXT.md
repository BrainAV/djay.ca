# 🎵 DJAY.ca - Project Context

## Overview
DJAY.ca is a standalone sub-brand and the central hub for all audio software and DJ services created by the [BrainAV organization](https://github.com/orgs/BrainAV/). The site is evolving to maintain its distinct identity as a comprehensive platform that not only provides professional DJ tools but also serves as a premier destination for discovering and listening to internet radio—positioning itself as a modern alternative to platforms like TuneIn, while still functioning under the broader BrainAV tech umbrella.

## Product Ecosystem
1. **Radio Stream Player (`radio.djay.ca`)**: Powered by the **Radio-Stream-Player-PHP** repository. Includes user accounts, station management, and a robust SQL backend. Currently operating as a standalone application.
2. **DJ MixKit (`mixkit.djay.ca`)**: A powerful browser-based DJ software featuring dual decks, effects, and mixing capabilities, requiring no installation.
3. **Jukebox App (In Development)**: A "Touch Tunes"-style collaborative jukebox experience, currently being prototyped by BrainAV.
4. **DJ & Event Planner Pro (Coming Q2 2026)**: A comprehensive CRM and event management tool for professional DJs.
5. **Professional DJ Services**: Booking and information for Jason Brain and a curated network of professional DJs across Canada.

## Architecture & Style
- **Tech Stack**: Vanilla HTML, CSS, and JavaScript.
- **Design Aesthetic**: Modern, responsive, dynamic styling. Emphasis on engaging visuals, such as glassmorphism, animated backgrounds, and clean typography.
- **Hosting**: GitHub Pages via the custom domain `djay.ca`.

## Core Objectives for Current Revamp
- Elevate the **Radio Stream Player** to be a marquee feature on the homepage, highlighting its capabilities as a TuneIn competitor.
- Clearly present the dual-nature of the platform: A destination for music listeners (Radio) AND a toolset for creators (DJ Apps).
- Ensure navigation and cross-linking between the main site, the Radio Player, and DJ MixKit are seamless.
