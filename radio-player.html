Radio Stream Player

A beautiful, highly customizable internet radio portal. Listen to curated global streams or add your own ad-free.

Launch Radio View on GitHub

📻 Custom Stations

Start with our curated electronic music streams from DI.FM and others, or easily add your own HTTP/HTTPS stream URLs to build a massive personal library.

✨ Live Metadata Integration

Experience real-time track, artist, and show information. Data is pulled directly from Icecast/Shoutcast streams and securely synced to your OS lock screen via the MediaSession API.

📊 Web Audio Visualizers

Transform your listening experience with beautiful, hardware-accelerated audio visualization. Built entirely using vanilla JS Canvas and the Web Audio API with multiple distinct drawing styles.

🛡️ Secure Edge Proxy

Natively bypass restrictive browser "Mixed Content" warnings. We use a custom Cloudflare Workers edge network proxy to securely wrap standard HTTP radio streams into HTTPS.

Open Source Architecture

The Radio Stream Player is proudly open-source and built with vanilla web technologies to remain lightweight and lightning-fast. The architecture revolves around a robust Audio element wrapper capable of auto-reconnecting dropped streams, handling cross-origin (CORS) constraints dynamically, and exposing raw frequency data to our visualization canvas engine.

Have a feature request or want to contribute new visualizer styles? Head over to our GitHub Repository.
