Radio Stream Player

A beautiful, highly customizable internet radio portal. Listen to curated global streams or add your own ad-free. Now updated to v2.x.x with user accounts, persistent settings, and a massive global directory.

Launch v2.1.x (Stable) Launch Legacy (v1.4.x) GitHub Repo

📻 Global Directory

Access over 30,000 live radio stations natively from the integrated Radio Browser API. Search by genre, country, or language and build your ultimate library.

🔐 Personal Accounts

New PHP & MySQL backend integration allows you to create a secure account, sync your favorites across devices, and customize your personal player settings.

📊 Web Audio Visualizers

Transform your listening experience with beautiful, hardware-accelerated audio visualization. Choose from multiple styles including VU meters, spectrum bars, and neon waves.

🛡️ Secure Edge Proxy

Natively bypass restrictive browser "Mixed Content" warnings. We use a custom Cloudflare Workers edge network proxy to securely wrap standard HTTP radio streams into HTTPS.

A Multi-Version Evolution

The Radio Stream Player has evolved into a robust web application. While v2.x.x (Stable) offers a complete platform with user management and databases, we continue to host v1.4.x (Legacy) at radio1.djay.ca for those who prefer the lightweight, standalone vanilla experience.

Have a feature request or want to contribute new visualizer styles? Head over to our GitHub Repository.
